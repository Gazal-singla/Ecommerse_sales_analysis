# E-Commerce Sales Analysis

## Overview
Analyzed 100K+ orders from Brazilian E-Commerce dataset using SQL and Excel.

## Tools Used
- MySQL (SQL queries, CTEs, Window Functions)
- Microsoft Excel (Dashboard, Charts)

## Key Insights
- Credit card contributes 73%+ of total revenue
- Bed & Bath is the top selling product category
- 97% customers are one-time buyers — retention opportunity
- November 2017 was peak revenue month (+53% growth)
- RR state has highest average delivery time (~30 days)

## Files
- queries/analysis_queries.sql — All SQL queries
- results/ecommerce_analysis.xlsx — Excel Dashboard
- dashboard.png — Dashboard Screenshot

## Dataset
Brazilian E-Commerce Public Dataset by Olist (Kaggle)
